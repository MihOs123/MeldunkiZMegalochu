Gracz może wystawić dowolny przedmiot na sprzedaż na rynku. 
Sprzedawca ustala cenę domyślną oraz ile może zejść z ceny (zniżka).
Kupujący może rzucić żeby się targować, odejmując umiejętność jakąś od sprzedawcy. Po rzucie nie można się wycofać

| Wynik         | Ile Płaci                  |
| ------------- | -------------------------- |
| Kryt. Sukces  | Cena domyślna - zniżka     |
| Sukces        | Cena domyślna - (zniżka/2) |
| Część. Wynik  | Cena domyślna              |
| Porażka       | Cena domyślna + (zniżka/2) |
| Kryt. Porażka | Cena domyślna + zniżka     |
